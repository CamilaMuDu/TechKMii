# TechKMii

Proyecto desarrollado en **C# WinForms** como parte de la materia *Programación III*.  
El sistema está orientado a la **gestión de eventos y cotizaciones**, integrando lógica de negocio con salida en XML y envío por correo electrónico.

## 🚀 Características principales
- Interfaz gráfica con formularios en WinForms.
- Serialización y deserialización en **XML/JSON**.
- Generación de cotizaciones en HTML y envío por correo.
- Manejo de eventos y arquitectura orientada a objetos.
- Configuración de reglas arquitectónicas para mantener el código limpio y escalable.

## 📂 Estructura del proyecto
- `Program.cs` → Punto de entrada de la aplicación.
- `Form1.cs` → Formulario principal con lógica de UI.
- `Properties/` → Configuración, recursos y ajustes del proyecto.
- `TechKMii.csproj` → Archivo de proyecto de Visual Studio.
- `TechKMii.sln` → Solución principal.

## 🛠️ Tecnologías utilizadas
- Lenguaje: **C#**
- Framework: **.NET**
- IDE: **Visual Studio**
- Control de versiones: **Git + GitHub**
